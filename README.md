## Requirement

- Java 8
- SpringBoot
- Node
- Angular2+


## Deploy Backend

To deploy the project Alticci_Sequence_Backend
 in the terminal, run 
```bash
  java -jar demo-0.0.1-SNAPSHOT.jar
```
## Deploy Frontend
To deploy the project Alticci_Sequence_Frontend
 in the terminal, run 
```bash
  ng serve
```

## Endpoint usado
* {n} is selected number
* HTTP method -> Post
```bash
  http:localhost:8080/alticci/{n}
```
